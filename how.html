How I do it

Every project is different because every user group has different needs.

My Process

  • Get to know the Users

    Research the industry
    Meet with the users
    Listen to the them
    Observe how they work
    Find out their likes & dislikes
    Ask them question
    Develop Personas

  • Set it up

    BrainStorm Flow
    Get to work with graph-paper
    Talk to the Devs
    Prototype - use kinestetics to see the big picture

  • Test it out

    Find the problem areas
    Let real users use it
    Listen to feedback
    Keep an open mind

  • Repeat

    Communicate Effectively
    Understand that no one person has the answer
    Know time spent now has a positive impact later
